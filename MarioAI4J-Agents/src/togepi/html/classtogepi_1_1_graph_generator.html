Detailed Description

Core Class that contains the Graph and must be instantiated before actionSelectionAI() in your agent class. The constructor takes two integers which correspond to the X and Y size of the grid you want to construct. Note, can be no larger than the perceptive grid, but can be smaller and non square e.g. 5x9. Call generateGraph(Entities a, Tiles b, AgentType T) in actionSelectionAI() to generate the grid ON FIRST RUN

Function call generate the graph. Takes 3 Arguments Current Entities, Current Tiles, and Agent-Type(Currently Only valid for A-Star). First creates all nodes in the given range (X by Y) Then links the nodes to their children/parents. Finally sets if a node is a goal state. Which currently is any node on the right edge of the graph that has no blocks or enemies.
